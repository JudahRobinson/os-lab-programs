PROCEDURE

Step 1: Download VMware

1. Open a web browser.
2. Visit:
https://www.vmware.com
3. Navigate to:
Products → Workstation Player
4. Download Linux bundle file.

Example:
VMware-Player-Full-17.x.x.bundle


Step 2: Open Terminal

Ctrl + Alt + T


Step 3: Navigate to Downloads Folder

cd Downloads

ls


Step 4: Make Installer Executable

chmod +x VMware-Player-Full-*.bundle


Step 5: Install VMware

sudo ./VMware-Player-Full*.bundle

Enter root password.


Step 6: Launch VMware

Open:
VMware Workstation Player


Step 7: Create New Virtual Machine

1. Click Create a New Virtual Machine
2. Select Installer disc image file (ISO)
3. Choose Ubuntu ISO file

Example:
ubuntu-22.04.iso


Step 8: Select Guest Operating System

Linux

Ubuntu 64-bit


Step 9: Name Virtual Machine

Ubuntu VM


Step 10: Allocate Disk Size

20 GB

Choose:
Store virtual disk as single file


Step 11: Configure Hardware

RAM : 2–4 GB

CPU : 2 cores

Disk : 20 GB

Network : NAT


Step 12: Start Virtual Machine

Click:
Power on this Virtual Machine


Sample Output

VMware installation completed successfully.

Ubuntu virtual machine started successfully.


Result

VMware Workstation Player installed and Ubuntu virtual machine created successfully.

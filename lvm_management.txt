PROCEDURE

Step 1: Install LVM Package

Ubuntu/Debian:

sudo apt install lvm2


Step 2: Create Physical Volume

sudo pvcreate /dev/sdb1


Step 3: Create Volume Group

sudo vgcreate my_vg /dev/sdb1


Step 4: Create Logical Volume

sudo lvcreate -L 2G -n my_lv my_vg


Where:

-L 2G → Logical volume size

-n → Logical volume name


Step 5: Format Logical Volume

sudo mkfs.ext4 /dev/my_vg/my_lv


Step 6: Create Mount Directory

sudo mkdir /mnt/lvm


Step 7: Mount Logical Volume

sudo mount /dev/my_vg/my_lv /mnt/lvm


Step 8: Verify LVM

Check physical volumes:

sudo pvdisplay


Check volume groups:

sudo vgdisplay


Check logical volumes:

sudo lvdisplay



Implementation Code

sudo apt install lvm2

sudo pvcreate /dev/sdb1

sudo vgcreate my_vg /dev/sdb1

sudo lvcreate -L 2G -n my_lv my_vg

sudo mkfs.ext4 /dev/my_vg/my_lv

sudo mkdir /mnt/lvm

sudo mount /dev/my_vg/my_lv /mnt/lvm

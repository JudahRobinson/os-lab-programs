PROCEDURE

Step 1: Check Existing Swap

swapon --show

or

free -h


Step 2: List Available Disks

sudo fdisk -l


Step 3: Create a New Partition

sudo fdisk /dev/sda


Inside fdisk:

n  → Create new partition
p  → Primary partition
3  → Partition number

Press Enter for First sector

+2G → Last sector size


Step 4: Change Partition Type to Swap

Inside fdisk:

t
82

Type 82 represents Linux swap.

Save changes:

w


Step 5: Format Partition as Swap

sudo mkswap /dev/sda3


Step 6: Activate Swap Partition

sudo swapon /dev/sda3


Step 7: Verify Swap

swapon --show

or

free -h



Implementation Code

sudo fdisk /dev/sda

sudo mkswap /dev/sda3

sudo swapon /dev/sda3

swapon --show

free -h



Sample Input

Partition size : +2G

Partition type : 82

Device name : /dev/sda3



Sample Output

NAME        TYPE       SIZE USED PRIO
/dev/sda3   partition   2G   0B   -2


Output of free -h:

Swap: 2.0G 0B 2.0G
